# Sistema de Chamados (Flask)

Projeto desenvolvido em Python utilizando Flask, com o objetivo de simular um sistema simples de gerenciamento de chamados.

## 🚀 Funcionalidades

- Criar chamados
- Listar solicitações
- Atualizar status (Aberto, Em andamento, Fechado)
- Excluir chamados
- Organização por setor
- Persistência de dados com JSON

## 💻 Tecnologias

- Python
- Flask
- HTML / CSS
- JSON

## 🎯 Objetivo

Praticar lógica de programação e dar os primeiros passos no desenvolvimento web com Flask.
